Bem-Vindo

Este é o projeto de Composição Web do estudante de Engenharia Informática:

  • Luis Henrique Cunha Rodrigues, nº 42878.


Este site tem como objetivo mostrar algumas noções básicas de diversos tipos de jogos. A DTJI (Don't Think Just Imagine) é uma pequena empresa Brasileira de jogos indie, formada por um grupo de 5 pessoas com o mesmo sonho.


Futuramente este site poderá se tornar um site para notícias de jogos além da divulgação dos nossos jogos.
